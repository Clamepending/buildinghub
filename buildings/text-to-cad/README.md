# Text to CAD Building

BuildingHub snapshot for the open source Text to CAD harness.

- Source repo: https://github.com/earthtojake/text-to-cad
- Reviewed source commit: `710655fdbf8b6b0ae0f13a9a371ab3b5debab1f7`
- Category: Design
- Trust lane: helper-command

This entry is declarative metadata for search, setup guidance, Agent Town placement, and generated agent guides. It does not install the harness, run CAD generation, grant model credentials, or store generated CAD artifacts by itself.

Agents should use the generated Text to CAD building guide before operating the harness. The guide points them to the upstream `AGENTS.md`, bundled CAD skill, bundled URDF skill, CAD Explorer viewer, and explicit generation commands.
